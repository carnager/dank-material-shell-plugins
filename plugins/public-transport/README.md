# Public Transport Plugin

Dank Material Shell widget for:

- favorite station departures
- start and destination lookup
- upcoming journey suggestions

Settings accept either a plain station query, a station id, or `Label|StationId`.

Examples:

- `Berlin Hbf|de-DELFI_de:11000:900003201`
- `Berlin Suedkreuz`
- `de-DELFI_de:05954:8174`
